


		# Github를 사용하기 위한 Gitbash- 명령어

	1. 폴더를 만든다.
	
	2. 그 폴더랑 자신의 Repositories랑 연결시켜준다.
	
	3. 새로만든 폴더에 .git을 만들어 준다. => git init
	
	4. 그 다음 연결 시켜준다. =>  git remote add origin repository address(주소 붙여넣기)
	
	5. git pull origin master : 지정 해 놓은 Repository에서 자료를 로컬로 가져옴.

	6. git status : 현재 Git 과의 상태를 Check

	7. git add . : 변경된 파일들을 모두 Tracking !

	8. git commit -m "message" :  인식할 수 있는 commit을 달아준다. (git add . 다음 동작)

	9. git push origin master : 실제로 Git에 PUSH!
